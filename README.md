# ❄️ WINTER ANALYTICS | v1.0
**Predictive AI Modeling & Quantitative Sports Intelligence**

---

## 🚀 EXECUTIVE SUMMARY
WINter Analytics is a proprietary sports betting terminal built for high-stakes quantitative analysis. By integrating real-time global news scrapers with Google Gemini 2.5 Flash AI, the platform translates raw sports data into actionable betting intelligence. This prototype is architected to demonstrate a seamless user journey from KYC onboarding to advanced tactical simulation.

---

## 🛠️ THE TECHNICAL STACK
* **Interface:** High-fidelity Glassmorphism UI (Vanilla HTML5, CSS3, ES6 JavaScript).
* **AI Engine:** Google Gemini 2.5 Flash via Make.com (Real-time sports event analysis).
* **Backend:** Airtable Serverless CMS (Multi-sport data warehousing).
* **KYC Module:** AI OCR Integrated Document Parsing via `ocr.brubai.net`.
* **Visual Data:** Dynamic Vector Charting Engine (Chart.js) for team performance ratings.
* **Animation:** Kling 3 & Seedream 5 AI-generated cinematic sports highlight reels.

---

## 🧠 CORE FEATURES

### 1. Multi-Sport Intelligence Feeds
Live data synchronization across seven global sports categories:
* **Basketball:** NBA, PBA, EuroLeague, CBA, KBL, Japan B League
* **Football:** Global Soccer (Premier League, La Liga, MLS).
* **Motorsports:** F1, NASCAR, MotoGP, Indycar
* **Additional:** Combat Sports, US Majors, Club & Court, and Esports.

### 2. Advanced AI Tactical Scouting (under construction)
A premium module that performs a "Global Sweep" of the highest-leverage matchups. 
* **Performance Vectors:** AI calculates 1-100 scores for Striking, Defense, Pace, Momentum, and Coaching.
* **Win Probability:** Real-time percentage calculation based on sharp-money momentum.

### 3. Integrated KYC Onboarding
A production-ready registration flow. Users are routed through a dedicated OCR terminal to parse government IDs and perform face-matching before gaining terminal access.

### 4. Interactive Financial Dashboard
* **Portfolio Tracking:** Live profit/loss visualization with SVG sparkline charts.
* **Authenticated Access:** Secure login flow (Demo: `brub` / `brub1!`).
* **Payment Silhouettes:** Pre-integrated silhouettes for Visa, Mastercard, PayPal, and PayMongo.
